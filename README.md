# 密码计算器
根据SOP时间和当前时间自动计算8155动态密码、8255新旧版动态密码、工程模式动态密码。

## 功能
- 支持自定义SOP时间（日历选择/手动输入）
- 自动计算4种动态密码
- 每小时自动更新密码
- 适配移动端/PC端

## 使用方法
1. 克隆仓库：`git clone https://github.com/LonelyJaker/password-calculator.git`
2. 打开 `password-calculator.html` 文件即可使用

## 计算规则
- 8155&8255动态密码 = (SOP时间 × 当前时间(MMDDHH)) 的最后六位
- 8255新版动态密码 = (SOP时间 × 当前时间 - 当前小时) 的最后六位
- 8255旧版动态密码 = (旧版SOP时间 × 当前时间 - 当前小时) 的最后六位
- 工程模式动态密码 = SOP时间×当前时间-当前小时，取最后六位
